## TetrisJS

Simplemente ejecutar:

1. `pnpm install` o `npm install`

Arrow Up => Rotar pieza

Arrow Down => Bajar pieza

Arrow Left/Right => Mover eje en x

Tiempo de descenso de las piezas por defecto: 1 segundo.
